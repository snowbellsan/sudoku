あくまで個人用にプレイするために作った、数独パズルです。

一応難易度選択機能がついてます。

広告とか何も出ないので、気軽にプレイできます。

## Live Demo

[ここから実行](https://snowbellsan.github.io/sudoku/)

遊び方：

難易度を選んだ後に「スタート」を押すことでパズルが生成されます。任意のマスを選んでから下部にあるパネルから数値を選んで、マスに数値を入力します。

メモをONにすると、マスに簡易メモが入力できます。


ヒントをONにすると、間違った数字を入力すると赤字で教えてくれます。
これによって当てずっぽうでもクリアできます…。
また、不要になったメモが消えるようになります。
